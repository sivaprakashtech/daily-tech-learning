# Software Testing Life Cycle (STLC)

## What is STLC?

STLC (Software Testing Life Cycle) is a sequence of activities performed by the testing team to ensure software quality.

## Phases of STLC

1. Requirement Analysis
2. Test Planning
3. Test Case Development
4. Test Environment Setup
5. Test Execution
6. Defect Reporting
7. Test Closure

## Interview Question

**Q:** What are the phases of STLC?

**Answer:**
Requirement Analysis → Test Planning → Test Case Development → Test Environment Setup → Test Execution → Defect Reporting → Test Closure
